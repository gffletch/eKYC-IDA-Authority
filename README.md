# README.md for eKYC-IDA-Registry-Profile

### What is this repository for? ###

* This is a repository for OpenID Foundation's [eKYC and Identity Assurance WG](https://openid.net/wg/ekyc-ida/).
* The document(s) are written in [markdown](https://bitbucket.org/tutorials/markdowndemo) and translated to html using [mmark](https://github.com/mmarkdown/mmark)

### Current activity in this repository

* Development of specifications relating to Delegated Authority for a range of use cases with metadata about the processes behind the 
* Specifications are anticipated to include at least a document describing a generalised representation of delegation between parties fo various types.
* Parties that may be involved in a delagation of authority may include a natural person, a legal entity, a workload, or an AI agent

### Contribution guidelines ###

* There are two ways to contribute, creating issues and pull requests in this Github repository
* All proposals are discussed in the WG via the mailing list and in our regular calls before being accepted and merged.
* If you wish to actively participate then please review and sign the [OpenID Foundation Contribution Agreement](https://openid.net/intellectual-property/openid-foundation-contribution-agreements/)
* Also please review [The OpenID Foundation (OIDF) Antitrust Policy](https://openid.net/wp-content/uploads/2024/09/OIDF-Antitrust-Policy_Final_2024-09-12.docx) 
* and other policies available at https://openid.net/policies/

### Who do I talk to? ###

* The WG can be reached via the mailing list openid-specs-ekyc-ida@lists.openid.net
* Regular eKYC & IDA WG calls are in the OpenID Foundation Calendar at https://openid.net/calendar/
